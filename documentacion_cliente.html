Bienvenido a la Plataforma de Bomberos B-120

Esta guía te ayudará a entender y usar la plataforma de manera sencilla. Si tienes dudas, contacta al administrador.

¿Cómo acceder?

1. Página principal:
Ir a la página principal
2. Iniciar sesión o registrarse:
Usa los botones de Login o Registro en la parte superior derecha.
3. Panel de administración:
Si eres admin o superadmin, accede a /admin/.
Si tienes problemas para acceder, contacta al soporte para restaurar tus permisos.

Documentación de la API

Swagger (interactiva): /api/swagger/
Redoc (visual): /api/redoc/

Glosario

API: Puente para que otras aplicaciones se conecten con la plataforma.
Admin: Usuario con permisos para gestionar la plataforma.
Superadmin: Usuario con todos los permisos, puede crear otros admins.

Ayuda rápida

  • ¿No puedes acceder? Pide al soporte que te asigne permisos de admin o superadmin.
  • ¿No entiendes la API? Usa la guía visual y los ejemplos en Swagger y Redoc.
  • ¿Problemas con login/registro? Verifica tu usuario y contraseña, o solicita restablecimiento.
